# 🧠 Deep Learning Projects

A collection of beginner-friendly Deep Learning projects built using **TensorFlow** and **Keras**. This repository demonstrates the implementation of Artificial Neural Networks (ANNs) for regression, binary & multi-class classification, and Natural Language Processing (NLP).

The projects focus on understanding the complete deep learning workflow, from data preprocessing and model building to performance evaluation.

---

## 🚀 Projects

### 📈 1. Regression using ANN
- Built an Artificial Neural Network for a regression problem.
- Loaded datasets directly from TensorFlow/Keras libraries.
- Performed data preprocessing and feature scaling.
- Trained and evaluated the model using regression metrics.

### 🎯 2. Classification using ANN
Implemented ANN models for:
- Binary Classification
- Multi-Class Classification

Key highlights:
- Loaded datasets directly from TensorFlow/Keras libraries.
- Detailed explanation and implementation of **Activation Functions**.
- Compared multiple activation functions and their use cases.
- Built, trained, and evaluated ANN models.
- Achieved good classification accuracy.

### 💬 3. NLP using ANN
- Built a text classification model using TensorFlow and Keras.
- Loaded datasets directly from TensorFlow/Keras libraries.
- Performed text preprocessing and tokenization.
- Trained the ANN model and evaluated its performance using accuracy metrics.
